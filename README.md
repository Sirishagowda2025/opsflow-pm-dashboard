# OpsFlow — AI-Powered Project Operations Dashboard

Live project management tool built to simulate Google Operations 
Center (GOC) workflows.

🔗 **[Try it live →](https://steadyflow-desk.lovable.app)**

---

## What it does

- Task management with priority, deadlines, and risk flagging
- Dedicated risk tracker — surfaces all flagged tasks instantly
- AI-powered report generation — reads live tasks and produces 
  a weekly stakeholder update, risk summary, and recommendations
- Real-time project health — completion %, overdue count, risk count

---

## Why I built this

I'm applying for the Google Operations Center PM Apprenticeship.
Every feature in OpsFlow maps directly to a responsibility listed 
in the GOC JD:

| GOC Requirement | OpsFlow Feature |
|---|---|
| Distill data for actionable takeaways | AI report reads live task data |
| Identify and manage project risks | Risk flag + Risks page |
| Share documentation with stakeholders | Copy report + CSV export |
| Track project scope and outcomes | Deadline tracker + health % |
| Data-backed recommendations | Report generates 3 specific actions |
| Coordinate internal resources | Task assignment + status tracking |
